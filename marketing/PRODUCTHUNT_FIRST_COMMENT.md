Hey Product Hunt 👋

I’m building **Content Repurposer** — a CLI that turns a single blog post into **5 ready‑to‑post formats** in ~60 seconds (Twitter thread, LinkedIn post, Newsletter, TikTok script, Facebook post).

**Why I built it**
I was spending 2–3 hours reformatting each post. It felt like busywork, so I built a tool that does it in one command with quality scoring + auto‑retry if it’s not good enough.

**Who it’s for**
- Indie hackers / solo founders
- Developers shipping content without a team
- Anyone who writes long‑form but needs distribution

**What’s inside**
- 5 formats in one run
- Quality score (1–10) + auto‑retry if < 7
- Works with Markdown / HTML / plain text
- Runs locally (fast + private)

**Pricing**
- **CLI (self‑hosted): Free** — bring your own Gemini API key
- **Web app (coming soon): $29/month** (no install)

**Ask**
I’d love feedback on output quality and the CLI UX. If you try it, tell me what format you care about most — I’ll prioritize improvements there.
